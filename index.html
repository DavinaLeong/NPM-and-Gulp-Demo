Here are some 'dummy' Bootstrap componenets to test proper loading of resources...

WARNING! Requires extensive use of the command line / terminal.

Exercise Goals

  1. Init and install NPM
  2. Install packages via NPM:
    • jQuery 2.1.4
    • Bootstrap
    • Font Awesome
  3. Install Gulp CLI
  4. Install dev packages via NPM:
    • Gulp
    • Del
    • Gulp Minify CSS
    • Gulp Concat
  5. Run-once Gulp tasks
  6. Looping Gulp task (Gulp watch)

Notable commands:

NPM

Install a local package:

npm install --save|--save-dev <package-name>@<version.number>

Install a global package:

npm install -g <package-name>@<version.number>

Shorthand:

npm i --save|--save-dev <package-name>@<version.number>

Other common commands:

npm uninstall|un <package-name> - uninstall a package

npm ls - lists all installed packages

Gulp

Run a Gulp task:

gulp <task-name>

Note: the task-name can be omitted for a 'default' named task.
